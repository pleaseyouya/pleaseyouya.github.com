Remove Duplicates from Sorted Array

题目：从一个有序数组中删掉重复的数字。

例子：
input: [1,1,2]
output:[1,2]

一开始, input的指针指向第一个数。output为空。

因为第一个数一定不会被删。所以, output的指针指向1，然后input完成第一个数，指向第二个数。

因为input当前指向的数是1，output指向的数也是1，所以input当前指向的数要被删掉。
所以，input指针指向数字2，output指针仍然指向1。

此时，input指针指向的2不等于output指针指向的1，所以，ouput增加2，指针指向2。

所以，假设input指针是input_ptr，output的指针是output_ptr，
如果input_ptr指向的数等于ouput_ptr指向的数，则删掉input_ptr指向的数，input_ptr++，output_ptr不变。
否则，input_ptr指向的数添加到output，并且output_ptr更新到最新的数。
```
int deleteDuplicate(int a[], int n) {
int input_ptr = 1, output_ptr = 0;
for (; input_ptr < n; ++input_ptr) {
if (a[input_ptr] != a[output_ptr]) {
a[++output_ptr] = a[input_ptr];
}
}
return output_ptr + 1;
}
